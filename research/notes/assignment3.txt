Overview
This assignment will give you hands-on practice with Jupyter AI, reinforcing the three interaction modes covered in Class 6: the chat interface, the %%ai magic command, and programmatic looping. You will apply these tools to a topic relevant to your own research area.
Instructions
Complete all three parts below in a single Jupyter notebook. You can use the GW JupyterHub at https://go.gwu.edu/jupyter. Save your notebook with all outputs visible and submit it to Box (name the file with your own name).
Part 1: Chat Interface Exploration
Use the Jupyternaut chat interface to get help with a piece of code related to your research. This could be code you are currently working on, or a simple example you create for this assignment.
1. Create a code cell with at least 5 lines of Python code relevant to your research (data processing, numerical computation, visualization, etc.).
2. Drag the cell into the Jupyternaut chat and ask it to explain what the code does.
3. Ask a follow-up question requesting a modification or improvement to your code (e.g., add error handling, optimize performance, add a plot).
4. Insert the suggested code into your notebook and verify it runs correctly. (Ask follow-up questions if needed.)
Deliverable: In a markdown cell, briefly describe: (a) the original purpose of your code, (b) what modification you requested, and (c) whether the AI suggestion worked on the first try or required iteration.

Tips

• Use %ai reset to clear conversation history between experiments.
